événements- Performance optimisée : Séparation claire entre écriture et lecture- Recherche avancée : Full-text search sur les événements- Analytics temporelles : Analyse des tendances dans le temps- Scalabilité : Possibilité de scaler indépendamment### Contexte GyroscopsDans notre écosystème User → Organization → Workflow → Cloud Resources → Billing, Event Sourcing + CQS avec ElasticSearch est idéal pour :- Système de facturation : Audit trail des paiements avec recherche avancée- Logs d’application : Historique complet avec analytics- Métriques de performance : Traçabilité des performances dans le temps- Workflows : Historique des étapes avec recherche et analytics## 🏗️ Architecture Event Sourcing + CQS### Séparation des Responsabilités#### Command Side (Write)- Command Handlers :
